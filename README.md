# Premium Calculator

A sleek, modern calculator web app with multiple themes and keyboard support.

## Features

- 4 beautiful themes — Midnight, Sunset, Cyberpunk, Glacial
- Glassmorphism UI design
- Keyboard support
- Floating point precision handling
- Responsive layout

## Themes

| Theme | Description |
|-------|-------------|
| Midnight | Dark blue with cyan accents |
| Sunset | Deep rose with warm orange |
| Cyberpunk | Black with neon green/pink |
| Glacial | Light blue, clean and minimal |

## Getting Started

Just open `index.html` in your browser — no setup needed.

```bash
git clone https://github.com/yourusername/premium-calculator.git
cd premium-calculator
open index.html
```

## Project Structure

```
├── index.html
├── styles.css
└── script.js
```

## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `0–9`, `.` | Input numbers |
| `+`, `-`, `*`, `/` | Operators |
| `Enter` or `=` | Calculate |
| `Backspace` | Delete last digit |
| `Escape` | Clear |
